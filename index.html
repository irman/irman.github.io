Yelo.my October 2018 - June 2020
Car Clubs & Communities

YELO.my (as in hello!) is a community and vehicle management portal for car enthusiasts (or any regular car owners). The aim is to be the single point of contact on anything related to cars; Car clubs, Insurance & Road tax renewal, News, Service centers and more.

I’m the sole developer of this project. I designed the database, built the backend API, and also implemented the entire front-end interfaces. This project features a discord/slack-like chat, VueJS SPA, PWA-enabled web app, social sign-in, auto RSS fetching, and multi-tier role-based system. The live chat system uses Vuex and websocket (with Pusher).

Laravel VueJS VueX SPA Pusher MySQL MongoDB Redis RSS Mailgun AWS Docker
Infinity.Money January 2018 - December 2018
The Infinite Payment Solutions

Infinity.Money aims to revolutionize the fintech industry by providing a simple white-label payment solution for merchants and rewards buyers/payee at the same time. It comes with built-in loyalty points, redemption & promo codes.

I built 90% of this project. A multipart ecosystem payment gateway featuring stripe-like seamless (popup) integration. Built with API-based Lumen, Laravel, Angular, StencilJS (Web Component), PHP-based composer package, and private Composer/NPM package repository.

Laravel Angular StencilJS MySQL Composer Mailgun JWT Docker Docker GitLab Continuous Integration
